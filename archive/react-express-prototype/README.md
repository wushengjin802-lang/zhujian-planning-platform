# React + Express Prototype Archive

该目录保留第一阶段早期原型，已从正式 `apps/` 工作区移出。

- `web`：React + Vite 静态/交互原型；
- `api`：Node.js + Express + pg 原型 API；
- 仅作为页面信息架构、接口形态和历史实现参考；
- 不再作为 PDD 正式第一阶段主线启动、构建或发布。

正式主线：

- `apps/web-vue`：Vue3 + Element Plus + Pinia + Vue Router；
- `apps/api-py`：FastAPI + SQLAlchemy + Pydantic + Alembic；
- `apps/api-py/app/worker`：Celery worker。
