# 项目中心功能 v2.0 起始版本说明

日期：2026-07-06  
版本：v2.0.0  
范围：PDD 5.2 项目中心页面

## 开发边界

本版本从工作台切换到项目中心开发，严格限定在项目中心页面及其必要后端聚合能力内。

本版本不开发以下内容：

- 报告正文在线编辑；
- GIS空间分析；
- 投资测算模型；
- 方案比选与方案收敛；
- 工作台新增能力；
- 平台级完整RBAC。

## 新增能力

### 1. 项目中心聚合

新增接口：

```text
GET /api/project-center
```

返回项目台账、指标、模板、用户、状态字典、密级字典和能力开关。

### 2. 项目详情

增强接口：

```text
GET /api/projects/{project_id}
```

返回项目基本信息、成员、里程碑、统计数据、初始化状态和可执行动作。

### 3. 新建项目向导

前端项目中心新增三步向导：

1. 基本信息；
2. 模板与密级；
3. 成员与里程碑。

新建项目后自动：

- 生成项目编号；
- 绑定模板版本；
- 设置密级；
- 添加创建人为项目负责人；
- 创建默认里程碑；
- 返回完整项目详情。

### 4. 成员与里程碑维护

新增接口：

```text
POST   /api/projects/{project_id}/members
DELETE /api/projects/{project_id}/members/{user_id}
POST   /api/projects/{project_id}/milestones
PATCH  /api/projects/{project_id}/milestones/{milestone_id}
```

### 5. 项目状态流转

新增接口：

```text
POST /api/projects/{project_id}/status
```

支持状态：

- 建档中；
- 进行中；
- 已关闭；
- 已归档。

带基础状态迁移约束，避免任意跳转。

### 6. 项目复制

新增接口：

```text
POST /api/projects/{project_id}/copy
```

支持选择是否复制：

- 成员；
- 里程碑；
- 模板与密级设置。

## 前端变更

主要文件：

```text
apps/web-vue/src/views/ProjectsView.vue
apps/web-vue/src/api/client.ts
apps/web-vue/src/types.ts
```

项目中心页面从简单卡片列表升级为：

- 项目指标；
- 项目搜索和筛选；
- 项目卡片台账；
- 项目详情抽屉；
- 基本信息编辑；
- 成员维护；
- 里程碑维护；
- 初始化状态检查；
- 项目复制；
- 关闭、归档、重新打开。

## 后续仍需推进

项目中心后续版本建议继续推进：

- 项目创建向导草稿保存；
- 项目成员权限在所有业务接口贯通；
- 项目密级影响下载、RAG、模型外发；
- 项目关闭/归档前完整门禁；
- 项目版本修订链；
- 里程碑逾期提醒与工作台联动；
- 项目编号规则配置化。
