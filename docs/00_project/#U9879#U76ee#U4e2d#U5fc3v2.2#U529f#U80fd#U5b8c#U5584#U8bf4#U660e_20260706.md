# 项目中心 v2.2 功能完善说明

## 范围边界

本版本仅继续完善 PDD 5.2「项目中心」页面能力，不扩展工作台、资料中心、报告编制、GIS、投资测算、方案比选或领导驾驶舱。

## 本次新增

1. 建档向导草稿保存、载入、删除。
2. 项目中心聚合返回当前用户建档草稿 `wizardDrafts`。
3. 新建项目时可关联 `draftId`，创建成功后草稿自动标记为“已使用”。
4. 新增地区/初始化规则 `project_region_rules`。
5. 内置通用初始化规则与政府投资项目规则。
6. 项目创建、复制和手动初始化包时按 `regionRuleId` 生成资料清单、事实框架、章节目录和成果项。
7. 初始化记录保存 `ruleId`、`ruleName`、`ruleVersion` 和预期初始化项数量。
8. 项目详情和新建向导支持地区、地区/初始化规则选择。

## 新增接口

```text
GET    /api/project-drafts
POST   /api/project-drafts
DELETE /api/project-drafts/{draft_id}
GET    /api/project-region-rules
POST   /api/project-region-rules
```

```text
project_wizard_drafts

```

## 仍保留边界

- 模板/地区规则迁移差异分析。
- 地区政策库接入。
- 草稿协作、审批和过期清理。
- 全量业务接口 RBAC。
- 项目修订链。
- 关闭后全模块只读。
