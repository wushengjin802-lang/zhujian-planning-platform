# 项目中心功能增强说明 v2.4

版本：2.4.0  
日期：2026-07-06

## 范围边界

本次只继续完善 PDD 5.2 项目中心，不扩展工作台、资料中心、报告编制、GIS、投资测算或方案比选。

## 本次新增

1. 迁移计划审批流：计划创建后进入“待审批”，支持审批通过和驳回。
2. 迁移应用门禁：迁移计划必须审批通过后才能应用。
3. 迁移回滚：应用前记录项目模板/地区规则指针，回滚时恢复指针。
4. 关闭后只读边界：项目关闭或归档后，项目中心基础资料、成员、里程碑、资料清单、初始化包等维护动作被阻断。
5. 前端操作边界：迁移计划表格按行级 actions 显示审批、驳回、应用、回滚。
6. 项目详情提示 readonlyReason，避免用户误以为操作失败。

## 明确不做

- 不删除迁移中补齐的资料、事实、章节和成果项。
- 不做模板正式发布审批。
- 不做归档包不可变版本。
- 不做全量业务接口 RBAC。
- 不做跨模块关闭后只读拦截。

## 后续建议

下一版 v2.5 可继续项目中心，优先做“关闭后跨模块只读拦截 + 迁移回滚影响分析 + 修订版成果关联”。
