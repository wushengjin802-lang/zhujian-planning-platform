# 工作台功能 v1.6 剩余功能收口说明

版本：v1.6.0  
日期：2026-07-06

## 范围边界

本次仅完善工作台页及其支撑接口，不新增报告正文审阅、GIS、投资测算、方案比选、领导驾驶舱等非工作台模块。完整平台级 RBAC 仍作为后续平台治理任务，不在本次范围内。

## 本次完成

1. **SLA自动升级与催办留痕**  
   对逾期和即将到期的工作项、审核任务，生成工作台事件和站内提醒，避免只停留在前端展示。

2. **审核会签门禁**  
   P0 审核任务要求 2 个会签，P1 审核任务要求 1 个会签；未满足会签数量时，后端阻止直接通过，工作台展示会签进度和不可通过原因。

3. **通知中心基础接口**  
   新增通知列表、项目/状态筛选，以及通知订阅偏好读取和更新接口。

4. **强制终止任务入口**  
   异步任务取消接口新增 `force` 参数。工作台对疑似卡住任务展示“强制终止”按钮，并将强制终止请求写入任务阶段日志。

5. **后端卡片级容错辅助**  
   工作台聚合增加 `cardErrors`，非关键卡片失败时可降级显示，并在健康状态中体现。

## 仍保留的边界

- 完整平台级 RBAC；
- 工作项复杂流转规则与SLA自动升级到具体组织岗位；
- 正式报告正文锚点级审核；
- WebSocket/短信/邮件等外部推送；
- Worker 内部安全停止点和强制终止确认；
- 领导专用驾驶舱。
