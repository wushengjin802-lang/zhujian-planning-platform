# 项目中心功能 v2.3 迁移差异与修订链说明

## 范围边界

本版本只完善 PDD 5.2 项目中心页面能力，不扩展资料中心、报告编制、GIS、投资测算、方案比选或工作台功能。

## 新增能力

1. 模板/地区规则迁移差异预览。
2. 迁移差异按资料清单、事实框架、章节目录、成果项四类展示。
3. 迁移预览输出影响项数量、风险等级和建议说明。
4. 迁移计划可保存目标模板、目标模板版本、目标地区规则和目标规则版本。
5. 应用迁移计划时更新项目模板/地区规则，并补齐初始化包新增项。
6. 应用迁移计划不会自动删除现有资料、事实、章节或成果项。
7. 项目修订链基础模型，保存修订号、父修订、原因、状态和基线快照。
8. 项目详情新增“迁移与修订”页签。
9. Checklist 更新为 v2.3。

## 新增接口

```text
GET  /api/projects/{project_id}/migration-preview
POST /api/projects/{project_id}/migration-plans
POST /api/projects/{project_id}/migration-plans/{plan_id}/apply
POST /api/projects/{project_id}/revisions
POST /api/projects/{project_id}/revisions/{revision_id}/close
```

## 新增数据表

```text
project_rule_migration_plans
project_revisions
```

## 仍保留边界

- 模板/规则迁移审批流；
- 初始化包版本自动迁移回滚；
- 修订版成果不可变归档；
- 全量业务接口 RBAC；
- 关闭后全模块只读；
- 正式项目版本发布链。
