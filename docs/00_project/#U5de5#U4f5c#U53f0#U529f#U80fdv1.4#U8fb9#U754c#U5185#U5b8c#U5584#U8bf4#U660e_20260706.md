# 工作台功能 v1.4 边界内完善说明

版本：v1.4.0  
日期：2026-07-06

## 版本号修正

根据版本顺延要求：

- 上一轮“工作台边界内继续完善”应按 v1.3 归档；
- 本轮继续开发版本号为 v1.4；
- 后续继续在 v1.5、v1.6 等版本上递增。

## 本次边界

本次仍严格限定在工作台功能范围内，不新增报告编制、GIS、投资测算、方案比选等非工作台业务功能。

## 本次完成内容

### 1. 工作项与审核任务 SLA 提示

- 为工作项和审核任务补充默认截止时间：
  - P0：1天；
  - P1：3天；
  - P2：7天；
  - P3：14天。
- Dashboard 聚合接口返回 `dueAt` 与 `dueStatus`；
- 前端在待办和待审核行内展示“截止 / 即将到期 / 已逾期”标签；
- 工作台风险提醒中聚合逾期和即将到期事项。

### 2. 行级动作权限边界

后端对工作项、审核任务返回 `actions` 字段，前端按权限展示操作按钮，避免用户看到不可执行动作：

- 工作项：领取、完成、转交、取消、意见、查看记录；
- 审核任务：分配、通过、退回、意见、会签、查看记录。

说明：本次仍不替代平台级完整 RBAC，只在工作台行级操作范围内收敛边界。

### 3. 疑似卡住任务留痕

- 工作台仍按任务更新时间识别疑似卡住任务；
- 新增后端留痕：同一任务首次被识别为疑似卡住时，写入 `task_events`，阶段为 `stuck_detected`；
- 前端任务中心可通过阶段日志查看卡住检测记录。

### 4. 版本号与文件命名

- 根包版本号更新为 `1.4.0`；
- 前端包版本号更新为 `1.4.0`；
- FastAPI OpenAPI 版本号更新为 `1.4.0`；
- Checklist 文件名更新为：`住建平台功能完成度Checklist_v1.4_20260706.xlsx`。

## 未扩展内容

以下内容仍不在本次版本中扩展：

- 工作项评论回复楼层；
- 工作项 SLA 升级策略和自动催办；
- 完整审核会签规则；
- 通知订阅配置和 WebSocket 推送；
- Worker 安全停止点；
- 平台级完整 RBAC；
- 单卡后端隔离失败。

这些能力应在后续版本中根据优先级继续补齐。
