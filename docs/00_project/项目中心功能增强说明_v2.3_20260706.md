# 项目中心功能增强说明 v2.3

## 版本边界

本版本仅增强 PDD 5.2 项目中心能力，不扩展资料中心、报告编制、GIS、投资测算、方案比选或成果归档模块。

## 本次新增能力

1. 模板/地区初始化规则迁移差异分析。
2. 迁移记录持久化。
3. 应用迁移时仅补齐新增初始化项，不自动删除已有业务数据。
4. 项目修订链基础模型。
5. 创建修订记录时保存项目快照。
6. 项目详情页新增“规则迁移与修订”页签。

## 新增接口

- `GET /api/projects/{project_id}/rule-migration-preview?targetRuleId=...`
- `POST /api/projects/{project_id}/rule-migrations`
- `POST /api/projects/{project_id}/rule-migrations/{migration_id}/apply`
- `GET /api/projects/{project_id}/revisions`
- `POST /api/projects/{project_id}/revisions`

## 数据表

- `project_rule_migrations`
- `project_revisions`

## 后续边界

仍未包含：地区政策库接入、完整规则迁移审批、项目修订正式发布链、修订成果版本、关闭后全模块只读和完整 RBAC。
