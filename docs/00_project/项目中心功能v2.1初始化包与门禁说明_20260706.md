# 项目中心功能 v2.1 初始化包与状态门禁说明

版本：v2.1.0  
日期：2026-07-06  
范围：仅限 PDD 5.2 项目中心页面功能，不扩展报告编制、GIS、投资测算、方案比选、工作台等模块。

## 一、本次目标

v2.0 已完成项目台账、详情抽屉、成员、里程碑、模板/密级、状态流转和项目复制。v2.1 在此基础上补齐项目中心的两个关键边界能力：

1. 完整项目初始化包；
2. 关闭/归档前置门禁。

## 二、新增能力

### 1. 项目初始化包

新增初始化包能力，创建/复制项目时自动补齐：

- 资料清单：必备/可选资料、状态、来源；
- 事实框架：项目基本信息、建设规模、投资测算等基础事实占位；
- 章节目录：项目概况、建设必要性、建设条件与需求分析、建设内容与规模、投资估算、结论建议；
- 成果项计划：Word、Excel、PPT、Archive；
- 初始化记录：记录初始化包版本、创建人、创建时间、创建数量摘要。

支持在项目详情中手动“生成/补齐初始化包”。重复执行不会重复创建已有同名项。

### 2. 资料清单维护

新增资料清单状态维护能力：

- 待上传；
- 已上传；
- 已确认；
- 不适用。

前端在“初始化包与门禁”页签中展示资料清单，并允许项目负责人或管理角色更新状态和来源说明。

### 3. 状态门禁

新增关闭/归档门禁：

- 关闭门禁检查：初始化包完整、无严重/阻断问题、无运行任务，并提示必备资料、章节审核等风险；
- 归档门禁检查：必须先关闭，且章节必须审核完成、成果项必须生成完成；
- 前端展示关闭/归档门禁卡片、阻断原因和提醒；
- 状态变更接口在门禁未通过时返回 409，并返回门禁明细。

## 三、新增 API

```text
POST  /api/projects/{project_id}/initialize
PATCH /api/projects/{project_id}/materials/{material_id}
GET   /api/projects/{project_id}/status-gate?targetStatus=已关闭
GET   /api/projects/{project_id}/status-gate?targetStatus=已归档
```

## 四、新增数据表

```text
project_material_requirements
project_initialization_records
```

新增迁移：

```text
apps/api-py/alembic/versions/20260706_0007_project_center_v21_initialization_gates.py
```

## 五、前端变化

项目中心详情抽屉新增“初始化包与门禁”页签：

- 初始化检查项；
- 待完善提示；
- 资料清单表；
- 关闭门禁卡片；
- 归档门禁卡片；
- 初始化记录列表。

## 六、未纳入本次边界

以下能力未在 v2.1 实现，后续版本继续推进：

- 建档向导草稿保存；
- 地区规则配置和绑定；
- 模板驱动的差异化初始化包；
- 初始化包版本迁移；
- 成员权限贯穿全部业务接口；
- 关闭后全模块只读；
- 项目修订链；
- 归档包不可变版本。
