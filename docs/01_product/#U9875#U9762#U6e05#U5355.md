# 页面清单

页面结构参考 `住建项目策划平台_静态演示版-v1.2`，第一阶段按 MVP 价值重排。

| 模块 | 页面 | 第一阶段处理 |
|---|---|---|
| 工作台 | 跨项目任务、风险、审核、运行状态 | 已完成增强版（实时聚合） |
| 项目中心 | 项目建档、成员、阶段、里程碑 | 开发 |
| 资料中心 | 资料上传、版本、解析、缺失清单 | 开发 |
| 事实与指标 | 事实抽取、确认、锁定、来源追溯 | 开发 |
| 编制工作台 | 章节树、正文、AI建议、引用、审核 | 开发 |
| GIS与投资测算 | 区位分析、投资测算、敏感性 | 轻量开发 |
| 方案决策与收敛 | 目标约束、候选方案、评价、冻结 | 保留入口 |
| 质量、审查与变更 | 质量规则、问题闭环、发布门禁 | 开发 |
| 成果中心 | Word、Excel、PPT、归档包 | 开发 |
| 知识与模板 | 政策、案例、模板、质量规则 | 开发 |
| 系统管理 | 角色权限、审计、模型网关、配置 | 基础开发 |
