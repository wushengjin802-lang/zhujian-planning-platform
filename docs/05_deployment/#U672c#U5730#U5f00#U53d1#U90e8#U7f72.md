# 本地开发部署

## 环境要求

- Node.js 22 或以上；
- npm 10 或以上；
- Python 3.12 或以上；
- PostgreSQL 16，数据库 `zhujian`，schema `public`；
- 可选：Redis、MinIO、LibreOffice。

## 启动

```bash
npm install
python -m pip install -r apps/api-py/requirements.txt
npm run api:migrate
npm run dev
```

## 端口

- Web：`5173`
- API：`8787`

## 说明

第一阶段已接入 PostgreSQL，连接配置位于 `.env`。当前使用 `zhujian` 数据库和 `public` schema，迁移工具为 Alembic。

文件对象存储支持 MinIO/S3，开发环境在 MinIO 不可用时使用本地 `storage/` 兜底。Celery 优先通过 Redis 派发任务，Redis 不可用时 API 会同步执行解析、质量检查和导出，保证 MVP 演示闭环不断。

## 数据库脚本

```bash
npm run api:migrate
python scripts/phase1_acceptance.py
```

## 可选基础服务

```bash
docker compose -f infra/docker/docker-compose.pdd.yml up -d
```

启动后可运行：

```bash
npm run worker:dev
```
